# 2021-08-01

## Objective

解了一題 LeetCode 需求是在同樣大小的陣列，在陣列為 0 的位置插入一個 0 ，主要的概念是先找到 0 的索引位置，再將該所有位置的元素都往下一個索引移動，時間複雜度是 O(n) linear time
